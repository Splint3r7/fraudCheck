# FraudCheck

## Requirements:

requirements.txt
```
colorama==0.4.1
urllib3==1.24.2
requests==2.21.0
beautifulsoup4==4.9.3
```

## installation:

```
pip3 install -r requirements.txt
```

## Usage:

For he list of domains
```
$ python3 fraud.py -l list.txt
```

for single domain
```
$ python3 fraud.py -s 10.0.0.10
```
